bootstrap-loader-example
=========================

```bash
### Install deps
npm i

### If doing hacking on the loader
# npm run install-local

### Run dev server with hot reloading
npm start
# Bootstrap 3
http://localhost:4000/bs3
# Bootstrap 4
http://localhost:4000/bs4

### Run production server
npm run bs:prod
# Bootstrap 3
http://localhost:4000/bs3
# Bootstrap 4
http://localhost:4000/bs4
```
